# Flow Talk Web 前端界面实现方案

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 基于 `flow-talk-server/docs/version` 的 v1-v7 能力，分阶段实现 Flow Talk Web 从认证入口到完整 IM 工作台的前端界面。

**Architecture:** 前端继续使用 React Router Framework Mode 做页面路由，Ant Design 做基础组件，`@mt-kit/request-axios` 做统一请求、鉴权和错误提示。界面按业务域拆分为 `auth`、`home/workspace`、`conversations`、`messages`、`devices`、`groups`、`presence`、`search` 等模块；每个模块包含自己的 API、类型、hooks 和组件，页面层只组合 view model 与展示组件。

**Tech Stack:** React 19, React Router 8 Framework Mode, TypeScript, Vite, Ant Design 6, `@mt-kit/request-axios`, Tailwind CSS 4.

Create `app/hooks/use-websocket-hook.ts`:

```ts
import {
  useEffect,
  useMemo,
  useRef,
  useState
} from "react";

import {
  getSession
} from "~/utils";
import type {
  IWsEvent
} from "~/utils/ws-event";

interface IUseWebSocketHookOptions {
  enabled: boolean;
  onEvent: (event: IWsEvent) => void;
}

interface IUseWebSocketHook {
  connected: boolean;
  sendEvent: (event: IWsEvent) => void;
}

function useWebSocketHook({
  enabled,
  onEvent
}: IUseWebSocketHookOptions): IUseWebSocketHook {
  const socketRef = useRef<WebSocket | null>(null);

  const [
    connected,
    setConnected
  ] = useState(false);

  useEffect(() => {
    const session = getSession();

    if (!enabled || !session?.token) {
      return;
    }

    const socket = new WebSocket(`ws://127.0.0.1:8080/ws?token=${session.token}&device_id=web`);

    socketRef.current = socket;

    socket.addEventListener("open", () => {
      setConnected(true);
    });

    socket.addEventListener("close", () => {
      setConnected(false);
    });

    socket.addEventListener("message", event => {
      onEvent(JSON.parse(event.data) as IWsEvent);
    });

    return () => {
      socket.close();
      socketRef.current = null;
    };
  }, [
    enabled,
    onEvent
  ]);

  return useMemo<IUseWebSocketHook>(() => {
    return {
      connected,
      sendEvent(event) {
        if (socketRef.current?.readyState === WebSocket.OPEN) {
          socketRef.current.send(JSON.stringify(event));
        }
      }
    };
  }, [
    connected
  ]);
}

export { useWebSocketHook };
export type {
  IUseWebSocketHook,
  IUseWebSocketHookOptions
};
```
